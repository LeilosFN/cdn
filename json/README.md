# CDN - JSON Configuration
Archivos de configuración dinámica y datos para la web y launcher.

## Archivos
- `status.json`: Estado de los servicios (Backend, Game, Web, etc.).
- `news.json`: Noticias mostradas en la web y launcher.

## Formato
Asegurarse de que el JSON sea válido antes de guardar. Un error aquí puede romper la web.
